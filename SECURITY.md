# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| 0.1.x (main) | Yes |

## Reporting a vulnerability

Please **do not** file a public GitHub issue for security reports.

Email the maintainer via GitHub: [@vikas-prasad-cx](https://github.com/vikas-prasad-cx) (use a private contact method linked on the profile, or open a draft security advisory on the repository when available).

Include:

- Description of the issue
- Steps to reproduce
- Affected module / version
- Any known mitigations

We will acknowledge receipt when possible and coordinate a fix before public disclosure.
